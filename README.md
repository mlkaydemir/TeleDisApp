# TeleDiş - Mobil Konsültasyon Uygulaması

TeleDiş, hekimlerin hastalarının diş fotoğraflarını inceleyebildiği, anamnez bilgilerine ulaşabildiği ve interaktif bir diş şeması üzerinden dental değerlendirme yapabildiği modern bir mobil uygulamadır.

## 🚀 Özellikler

- **Hekim Girişi**: Güvenli giriş paneli.
- **Hasta Paneli**: Yan menü üzerinden hasta listesi ve profil yönetimi.
- **Görüntülü Konsültasyon**: Hasta ağız içi fotoğraflarının detaylı incelenmesi.
- **Anamnez & Bilgi**: Hızlı erişim modalleri ile hasta geçmişi görüntüleme.
- **İnteraktif Değerlendirme**: 
  - İnteraktif diş şeması (Dental Değerlendirme).
  - Genel diş eti ve periodontal durum analizi.

## 🛠 Kullanılan Teknolojiler

- **React Native** (Expo SDK 52)
- **Android Studio** (Native Build)
- **React Navigation** (Stack & Drawer)
- **Vector Icons & Picker**

## 📦 Yerelde Nasıl Çalıştırılır?

Projeyi yerelde çalıştırmak için aşağıdaki adımları izleyin:

1. **Bağımlılıkları Yükleyin**:
   Terminalde projeyi açın ve kütüphaneleri yükleyin:
   ```bash
   npm install
   ```

2. **Native Projeyi Açın**:
   **Android Studio**'yu açın ve `android` klasörünü "Open" diyerek seçin.

3. **APK Oluşturma**:
   Android Studio menüsünden: 
   `Build > Build Bundle(s) / APK(s) > Build APK(s)`
   yolunu izleyerek debug APK dosyasını oluşturabilirsiniz.
